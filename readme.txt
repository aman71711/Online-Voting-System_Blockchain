Sure, here are the commands for installing the required dependencies:

Install Node.js: Download and install NodeJS from here.

Install Truffle and Ganache CLI using npm:

npm install -g truffle
npm install -g ganache-cli
Install Metamask browser extension: Download and install Metamask from here.

Once you have installed the dependencies, you can set up the development environment by following these steps:

Clone the repository: git clone https://github.com/aman71711/Online-Voting-System_Blockchain.git

Run a local Ethereum blockchain using Ganache CLI: ganache-cli

Configure Metamask on your browser and connect it to the local blockchain.

Deploy the smart contract to the local blockchain: truffle migrate 

Launch the development server: npm start

After launching the development server, you can access the application on your browser at http://localhost:3000.
